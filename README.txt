-------------------------------------------------------------------------------
                              Faroese G2P Models
-------------------------------------------------------------------------------

Authors         : Carlos Daniel Hernández Mena, Sandra Saxov Lamhauge,
                  Iben Nyholm Debess, Annika Simonsen.

Language        : Faroese.

Recommended use : speech recognition.

-------------------------------------------------------------------------------
Description
-------------------------------------------------------------------------------

The "Faroese G2P Models" is a set of two models trained with the software
tool "Sequitur-G2P" [1] which is a trainable grapheme-to-phoneme tool 
developed at RWTH Aachen University (https://www.rwth-aachen.de/) by 
Maximilian Bisani. One of the models is for Central Faroese and the other 
is for the East variant.

The set also includes two files with the corresponding repertory of phonemes
for the Central (central.phones) and the East (east.phones) variants.

Instructions of how to install and use Sequitur-G2P can be found in the
official Pypi repository. Just type in any web browser the keywords:
"sequitur-g2p pypi" to find it.
